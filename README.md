# pre-entrega
Servicios de Tarot y Coaching

¡Bienvenido/a al repositorio de mi proyecto de Front-End! Esta página web está diseñada para ofrecer servicios profesionales de lecturas de tarot y procesos de coaching, fusionando la guía intuitiva con el desarrollo personal.

El objetivo principal es brindar una interfaz intuitiva, armónica y profesional que permita a los usuarios conocer los servicios, agendar sesiones y explorar las diferentes propuestas.
